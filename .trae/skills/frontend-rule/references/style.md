# 样式规范

## CSS变量规则
命名格式：`--sk-{category}-{name}`
- 所有设计系统元素（颜色、字体、圆角、间距、阴影）统一使用CSS变量定义
- 禁止硬编码颜色、字号、圆角等值，必须引用已定义的CSS变量

## 样式编写规则
1. 组件样式必须使用`<style scoped>`隔离
2. 仅动态计算的样式可以使用内联样式，其他禁止
3. 统一遵循Apple风格设计系统：
   - 主色调：黑白灰为主，强调色使用Apple蓝(#0071e3)
   - 圆角：标准8px、卡片12px、圆形50%
   - 字体：标题用SF Pro Display，正文用SF Pro Text
