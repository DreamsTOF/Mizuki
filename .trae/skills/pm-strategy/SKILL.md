# 产品战略分析技能

> 整合Marty Cagan、Geoffrey Moore等战略大师的方法论，覆盖从市场分析到商业模式设计的全流程

## 核心能力
### 1. 战略分析框架
- **SWOT分析**：优势/劣势/机会/威胁四维度评估，制定进攻/防御/转型战略
- **波特五力模型**：行业竞争格局分析，评估供应商/买方/替代品/新进入者威胁
- **PESTLE分析**：政治/经济/社会/技术/法律/环境六大外部环境扫描
- **安索夫矩阵**：市场渗透/市场开发/产品开发/多元化四大增长策略选择

### 2. 商业模式设计
- **商业模式画布（BMC）**：9模块完整商业模式设计，覆盖价值主张、客户细分、渠道、收入来源等
- **精益画布**：创业项目专用的轻量化商业模式设计工具
- **价值主张画布**：针对细分客户的价值定位设计，现状→解决方案→改变后→替代方案四步结构

### 3. 产品战略规划
- **产品策略画布**：9模块产品战略设计，包含愿景、市场细分、价值主张、关键指标、增长策略等
- **机会解决方案树（OST）**：从北极星指标拆解机会点，探索解决方案的结构化工具
- **预死亡分析**：提前识别项目风险，提升30%问题发现率

## 输出模板
### SWOT分析模板
```
【SWOT分析报告】

**优势（S）**：
- [独特能力、品牌认知、技术优势等]

**劣势（W）**：
- [资源约束、技术限制、成本劣势等]

**机会（O）**：
- [市场增长、技术进步、竞品弱点等]

**威胁（T）**：
- [新兴竞品、需求变化、监管风险等]

**战略建议**：
- 进攻策略（S+O）：...
- 防御策略（W+T）：...
- 转型策略：...
```
