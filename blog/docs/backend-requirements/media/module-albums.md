# Albums 模块后端需求

## 概述

Albums（相册）模块用于管理博客中的相册集合。每个相册以独立目录形式组织，目录内包含相册元数据文件 `info.json`、封面图片（`cover.jpg` 或 `cover.webp`）以及相册内的所有图片文件。

## 数据模型

### 相册信息（AlbumInfo）

| 字段 | 类型 | 说明 |
|------|------|------|
| title | string | 相册标题 |
| description | string | 相册描述 |
| date | string | 相册日期 |
| location | string | 拍摄地点 |
| tags | string[] | 标签列表 |
| layout | "masonry" \| "grid" | 相册页面布局方式 |
| columns | number | 图片展示列数 |

### 相册结构

```
albums/
  {album-id}/
    info.json        # 相册元数据
    cover.jpg        # 或 cover.webp，相册封面
    image1.jpg
    image2.webp
    ...
```

### 图片信息

| 字段 | 类型 | 说明 |
|------|------|------|
| filename | string | 图片文件名 |
| url | string | 图片访问路径 |
| width | number | 图片宽度（像素） |
| height | number | 图片高度（像素） |
| size | number | 文件大小（字节） |
| mimeType | string | MIME 类型 |

## 功能点

### 1. 获取相册列表

- 扫描相册根目录，读取所有子目录作为相册。
- 从每个相册目录的 `info.json` 中读取相册信息。
- 返回相册基本信息列表，包含封面图片路径。
- 支持按日期排序。

**需要的数据：**
- 相册 ID（目录名）
- 相册标题
- 相册描述
- 相册日期
- 相册地点
- 标签列表
- 封面图片路径
- 图片数量

### 2. 获取单个相册详情

- 根据相册 ID 定位相册目录。
- 读取 `info.json` 获取相册完整信息。
- 扫描目录内所有图片文件，构建图片列表。
- 返回相册信息及图片列表。

**需要的数据：**
- 相册完整信息（title, description, date, location, tags, layout, columns）
- 图片列表（每张图片的 filename, url, width, height, size, mimeType）
- 封面图片标识

### 3. 创建相册

- 接收相册基本信息，创建新相册目录。
- 在目录内生成 `info.json`。
- 若上传了封面图片，保存为 `cover.jpg` 或 `cover.webp`。

**需要的数据：**
- 相册标题（必填）
- 相册描述
- 相册日期
- 拍摄地点
- 标签列表
- 布局方式（默认 masonry）
- 列数（默认 3）
- 封面图片文件（可选）

### 4. 更新相册信息

- 根据相册 ID 定位相册目录。
- 更新 `info.json` 中的字段。
- 支持单独更新封面图片。

**需要的数据：**
- 相册 ID
- 待更新的字段（title, description, date, location, tags, layout, columns 中的部分或全部）
- 新封面图片文件（可选）

### 5. 删除相册

- 根据相册 ID 定位相册目录。
- 删除整个相册目录及其内所有文件。

**需要的数据：**
- 相册 ID

### 6. 上传相册图片（支持批量上传）

- 根据相册 ID 定位相册目录。
- 接收一个或多个图片文件。
- 保存到相册目录内。
- 返回上传成功的图片信息列表。

**需要的数据：**
- 相册 ID
- 图片文件列表（每个文件包含文件名、二进制内容、MIME 类型）

### 7. 设置相册封面

- 根据相册 ID 定位相册目录。
- 将指定图片复制或重命名为 `cover.jpg`/`cover.webp`。
- 或接收新的封面图片文件并保存。

**需要的数据：**
- 相册 ID
- 目标图片文件名（从已有图片中设置）或新的封面图片文件

### 8. 删除相册中的单张图片

- 根据相册 ID 和图片文件名定位文件。
- 删除该图片文件。
- 若删除的是当前封面，需处理封面缺失情况（可清空封面或自动指定新封面）。

**需要的数据：**
- 相册 ID
- 图片文件名
